# i18n (Internationalization) Skill

다국어 지원을 위한 패턴과 구현 가이드라인입니다.

## 지원 언어

- **한국어 (ko)**: 기본 언어
- **영어 (en)**: 보조 언어

## 파일 구조

```
i18n/
├── ko.json    # 한국어 번역
├── en.json    # 영어 번역
└── index.ts   # 유틸리티 함수
```

## useTranslation Hook

```tsx
// hooks/useTranslation.ts
'use client';

import { useState, useCallback, useMemo } from 'react';
import ko from '@/i18n/ko.json';
import en from '@/i18n/en.json';

type Locale = 'ko' | 'en';
type Translations = typeof ko;

const translations: Record<Locale, Translations> = { ko, en };

// 로컬 스토리지 키
const LOCALE_STORAGE_KEY = 'image-puzzle-locale';

// 브라우저 기본 언어 감지
function getDefaultLocale(): Locale {
  if (typeof window === 'undefined') return 'ko';

  const stored = localStorage.getItem(LOCALE_STORAGE_KEY);
  if (stored === 'ko' || stored === 'en') return stored;

  const browserLang = navigator.language.toLowerCase();
  return browserLang.startsWith('ko') ? 'ko' : 'en';
}

export function useTranslation() {
  const [locale, setLocaleState] = useState<Locale>(getDefaultLocale);

  const setLocale = useCallback((newLocale: Locale) => {
    setLocaleState(newLocale);
    localStorage.setItem(LOCALE_STORAGE_KEY, newLocale);
  }, []);

  const t = useCallback(
    (key: string, params?: Record<string, string | number>): string => {
      const keys = key.split('.');
      let value: unknown = translations[locale];

      for (const k of keys) {
        if (value && typeof value === 'object' && k in value) {
          value = (value as Record<string, unknown>)[k];
        } else {
          console.warn(`Translation key not found: ${key}`);
          return key;
        }
      }

      if (typeof value !== 'string') {
        console.warn(`Translation value is not a string: ${key}`);
        return key;
      }

      // 파라미터 치환 ({{param}} 형식)
      if (params) {
        return value.replace(/\{\{(\w+)\}\}/g, (_, paramKey) => {
          return String(params[paramKey] ?? `{{${paramKey}}}`);
        });
      }

      return value;
    },
    [locale]
  );

  const toggleLocale = useCallback(() => {
    setLocale(locale === 'ko' ? 'en' : 'ko');
  }, [locale, setLocale]);

  return {
    t,
    locale,
    setLocale,
    toggleLocale,
    isKorean: locale === 'ko',
    isEnglish: locale === 'en',
  };
}
```

## TranslationProvider (선택적)

컨텍스트를 통한 전역 언어 관리가 필요한 경우:

```tsx
// providers/TranslationProvider.tsx
'use client';

import { createContext, useContext, ReactNode } from 'react';
import { useTranslation } from '@/hooks/useTranslation';

type TranslationContextType = ReturnType<typeof useTranslation>;

const TranslationContext = createContext<TranslationContextType | null>(null);

export function TranslationProvider({ children }: { children: ReactNode }) {
  const translation = useTranslation();

  return (
    <TranslationContext.Provider value={translation}>
      {children}
    </TranslationContext.Provider>
  );
}

export function useTranslationContext() {
  const context = useContext(TranslationContext);
  if (!context) {
    throw new Error('useTranslationContext must be used within TranslationProvider');
  }
  return context;
}
```

## 언어 선택 컴포넌트

```tsx
// components/LanguageSelector.tsx
'use client';

import { useTranslation } from '@/hooks/useTranslation';

export function LanguageSelector() {
  const { locale, setLocale, t } = useTranslation();

  return (
    <div className="flex items-center gap-2">
      <span className="text-sm text-gray-500">{t('settings.language')}:</span>
      <button
        onClick={() => setLocale('ko')}
        className={`px-2 py-1 rounded ${
          locale === 'ko'
            ? 'bg-blue-500 text-white'
            : 'bg-gray-200 hover:bg-gray-300'
        }`}
      >
        한국어
      </button>
      <button
        onClick={() => setLocale('en')}
        className={`px-2 py-1 rounded ${
          locale === 'en'
            ? 'bg-blue-500 text-white'
            : 'bg-gray-200 hover:bg-gray-300'
        }`}
      >
        English
      </button>
    </div>
  );
}
```

## 사용 예시

```tsx
// 컴포넌트에서 사용
import { useTranslation } from '@/hooks/useTranslation';

function GameTitle() {
  const { t } = useTranslation();

  return (
    <div>
      <h1>{t('game.title')}</h1>
      <p>{t('game.subtitle')}</p>
    </div>
  );
}

// 파라미터와 함께 사용
function CardCounter({ count }: { count: number }) {
  const { t } = useTranslation();

  return (
    <span>{t('game.play.cardsRemaining', { count })}</span>
  );
}
```

## 베스트 프랙티스

1. **키 네이밍**: 계층적 구조 사용 (`game.setup.uploadImage`)
2. **기본값**: 항상 한국어를 기본으로 작성
3. **일관성**: 같은 의미의 텍스트는 같은 키 사용
4. **파라미터**: 동적 값은 `{{param}}` 형식 사용
5. **타입 안전성**: TypeScript로 키 자동완성 지원 가능
